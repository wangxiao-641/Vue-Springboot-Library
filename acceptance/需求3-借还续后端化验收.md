# 需求变化 3：借书、还书、续借流程后端化——验收记录

验收日期：2026-07-16

## 改动范围

- 新增 `/circulation/borrow`、`/circulation/return`、`/circulation/renew` 三个业务接口。
- 新增 `CirculationService`，以事务统一维护 `book`、`bookwithuser`、`lend_record`。
- 前端借书、还书、续借每次操作只发送一个业务写请求，请求体仅包含 `readerId` 和 `isbn`。
- 新增 `verify_circulation_http.py` 黑盒验证脚本及运行文档。
- 修正 `dev.sh verify` 图书查询参数和中文 URL 编码。
