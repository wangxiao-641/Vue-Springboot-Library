## 四、权限边界验证

> 确认普通读者不能执行管理员专属操作。以下测试用读者账号的 token。

```bash
# 获取读者 token
TOKEN=$(curl -s -X POST http://localhost:9090/user/login \
  -H 'Content-Type: application/json' \
  -d '{"username":"reader","password":"123456"}' | \
  python3 -c "import sys,json; print(json.load(sys.stdin).get('data',{}).get('token',''))")

# 读者尝试删除图书（应该被拒绝）
curl -s -o /dev/null -w "%{http_code}" -X DELETE "http://localhost:9090/book/1" \
  -H "token: $TOKEN"

# 读者尝试查看所有读者列表（应该被拒绝或被限制）
curl -s -o /dev/null -w "%{http_code}" "http://localhost:9090/user?pageNum=1&pageSize=10" \
  -H "token: $TOKEN"
```
